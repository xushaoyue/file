## DICOM Editor
### 简介
DICOM Editor是一款专业的DICOM文件查看和编辑工具，专为满足医疗影像数据运维需求而开发。本软件由AI辅助制作，旨在解决传统DICOM编辑工具在标签定位和批量修改方面的痛点。
### 开发背景
在日常运维工作中，我们面临以下挑战：
•	现有DICOM工具难以快速定位需要修改的标签
•	大部分工具仅支持单值修改，无法满足批量修改需求
•	部门内部有频繁的DICOM文件查看需求
基于这些需求，我们开发了DICOM Editor，提供更高效、更便捷的DICOM文件处理体验。
### 主要功能
#### 🖱️ 拖拽打开
•	支持直接拖拽DICOM文件到软件界面
•	自动识别并打开DICOM文件
•	简化文件加载流程，提升工作效率
#### 📄 DICOM标签内容导出
•	将DICOM文件的所有标签内容导出为TXT文本文件
•	支持文件内容对比分析
•	提供便捷的标签检索功能
#### 🔍 DICOM标签检索
•	强大的标签搜索功能
•	支持按标签名称、标签ID等多种方式检索
•	快速定位需要修改的特定TAG
#### ⭐ 快捷修改
•	支持常用标签收藏功能
•	一键快速定位常用标签
•	提升重复性工作的处理效率
#### 📋 待修改列表
•	集中管理本次需要修改的所有标签
•	预览修改内容，确保修改准确性
•	批量应用修改，支持多值同时编辑
### 使用场景
#### 运维工作
•	批量修改DICOM文件元数据
•	快速修正标签错误
•	数据标准化处理
#### 日常查看
•	DICOM文件内容浏览
•	影像数据质量检查
•	元数据验证
### 安装与使用
#### 系统要求
•	Windows 10及以上操作系统
•	.NET Framework 4.8或更高版本
•	至少4GB内存
#### 安装步骤
1.	下载安装包
2.	运行安装程序
3.	按照向导完成安装
4.	启动DICOM Editor
#### 基本操作
1.	打开文件：拖拽DICOM文件到软件界面
2.	查看标签：在标签树中浏览DICOM文件内容
3.	搜索标签：使用搜索框快速定位特定标签
4.	修改标签：双击标签值进行编辑
5.	保存修改：点击保存按钮应用所有修改
### 注意事项
#### ⚠️ 重要提示：
•	修改DICOM文件可能影响医疗诊断结果，请谨慎操作
•	建议在修改前备份原始文件
•	仅授权人员应使用此工具进行文件修改
